# wordcards
##  專案簡介 
一個背英文單字的單字卡app!🤠🤠🤠

##  使用方式
左側單字列表
點擊一下：查看單字、英標、解釋，還可以點擊按鈕撥放單字念法
點擊兩下：開啟修改視窗，可以對單字、音標、音檔路徑、解釋作修改
